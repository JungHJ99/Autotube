# 국뽕 헤드라인 카피 + 권위 라벨 패턴 분석 (2026-05-22)

## TL;DR (5줄)

1. **국뽕 제목의 표준 골격은 "[인용/감정 키워드] + [충격 결과] + [진짜 이유 약속]"** 세 마디 구조다. 예: "한국에서 살고싶어 우는 중" + "일본인들 전부 다 한국行" + "이곳 가는 진짜 이유". 우리 트라이얼 제목은 이 셋 중 2개를 못 살리고 "왜?" 한 글자만 남겨서 클릭 트리거가 죽었다.
2. **자극어는 top-7 ("충격", "발칵", "경악", "통곡", "진짜 이유", "실제", "이러다") 만 반복**. 신작가/해외감동사연/엠빅뉴스 등 메이저 채널 4곳 21개 제목에서 70% 이상이 이 풀에서 1~2개를 쓴다.
3. **노란색 강조어는 의미 단위 — 국가명/숫자/결과 동사**. "1초", "66.2%", "통곡", "굴복", "한국行" 같은 4글자 이내 결과어가 색 분리되며 1글자 강조는 죽은 디자인.
4. **권위 라벨은 좌상단 `[해외감동사연]`, `[실제영상]`, `[자막뉴스]` 같은 4~6자 빨간 박스 + 한국어 한정**. 외국 학자/논문은 *제목 본문* 안에 "美 사회학자", "일본인 학자" 처럼 4~6자로 압축해서 끼워넣는 게 표준 — 라벨 박스 자체에는 거의 안 들어감.
5. **트라이얼 영상 (Itoi/Korea Journal/일본 여성) 의 진짜 hook 은 "일본인이 직접 쓴 한국행 논문"이다.** 인용형 + 의문 도발형 + 통계 폭격을 섞은 헤드라인이 압도적으로 유리. top-3 추천 조합은 본문 ### 추천 조합 섹션에 정리.

## 어순/구조 패턴 (top-7)

### 1. 인용 시작 + 통곡 결과 + 진짜 이유 (가장 강력, 본 영상 21개 중 9개)
구조: `"인용문" + 누가/어디서 + 결과 동작 + 진짜 이유`
- 예 #1: `"우리가 2류였다" + 미국 10대들 + 통곡하며 고백 + 충격적인 이유`
- 예 #9: `"한국에서 살고싶어 우는 중" + 일본인들 전부 + 한국행 + 이곳 가는 진짜 이유`
- 예 #15: `"최근 가장 충격적" + 미국 유명 교수 + 경악`
**왜 강력한가:** 1) 따옴표가 시각적으로 즉시 인용임을 알려주고 2) 통곡/우는/경악 같은 신체 반응 동사가 감정 즉시 전달하고 3) 마지막 "진짜 이유" 가 클릭 약속.

### 2. 발견형 라벨 + 본문 (해외감동사연 시그니처, 21개 중 5개)
구조: `[해외감동사연] 또는 [실제영상] + 본문 헤드라인`
- 예 #2: `[해외감동사연] 한국의 강경 대응법에 완전 굴복한 미국이 발칵 뒤집혔다!`
- 라벨 자체가 카테고리 인증 + 발견형 ("당신이 모르던 해외 사연") 신호를 동시에 함.

### 3. 통계 폭격형 (21개 중 4개)
구조: `숫자 + 동작 + 결과`
- 예 #3: `15만 명 귀국 폭주`
- 예 #4: `미국 중산층 80%가 한국 선택`
- 예 #20: `13억이 시청한 韓中日 비교`
- 우리 트라이얼 영상은 **66.2%** 라는 강력한 통계 보유 → 거의 무조건 활용해야 함.

### 4. 의문 도발형 ("진짜 이유는?")
구조: `(주어/사건) + 진짜 이유 / 그 이유 / 왜 / 이곳`
- 예 #7: `한국에선 힘 못쓰는 진짜 이유`
- 예 #9: `이곳 가는 이유`
- 예 #16: `美 유명 교수 경악한 이유`
- 우리 트라이얼이 이미 "그 진짜 이유" 를 쓰고 있는데 — **앞부분이 약하다**는 게 문제. 뒤만 강해도 클릭 안 일어남.

### 5. 권위 인용형 (인물의 발언 + 권위 직함)
구조: `직함/소속 + 동작 (경악/충격/발언)`
- 예 #15~18: `美 유명 교수`, `미국인 교수`, `美 사회학자`
- 예 #19: 일본 본인이 등장 — "이러다 한국에 다 뺏긴다"
- 직함 길이 4~7자가 최대 — "리츠메이칸大 사회학과 교수" 같은 12자는 죽음. "日 학자" / "日 교수" 가 한국 YouTube 의 표준.

### 6. 충격 표현 + 국가 (강력하지만 진부)
구조: `(주어) + 발칵/경악/충격/뒤집힌`
- 예 #5: `세계를 멈추게 한 충격 사태! 발칵 뒤집혔다!`
- 예 #11: `전세계 발칵ㅎㄷㄷ`
- 이미 인플레이션 — 단독으로 쓰면 가짜 국뽕 신호. 다른 구조 (인용 / 통계) 와 결합해야 함.

### 7. 비교 (vs / 차이) 폭로형
구조: `A vs B + 결과 충격`
- 예 #19: `이러다 한국에 다 뺏긴다`
- 예 #20: `韓中日 비교 분석`
- 우리 트라이얼은 한일 비교 본질 — 일장기 / 태극기 시각 분리 잘 활용 가능.

## 노란색 강조어 패턴

수집 영상의 썸네일 노란색 강조 단어들 (이전 보고서 + 본 분석 통합):

### 노란색이 자주 들어가는 단어 유형 (top-7)
1. **국가명** — "한국", "미국", "일본" (또는 한자 약자 "日", "韓", "美")
2. **숫자 결과** — "1위", "80%", "15만 명", "5명 중 4명"
3. **결과 동사 (1~2 음절)** — "통곡", "굴복", "발칵", "경악"
4. **시간 강조** — "1초", "단 3일"
5. **방향 표시** — "한국行", "→ Seoul"
6. **신원/직함** — "美 교수", "日 학자"
7. **의문어** — "이유?", "왜?" (단 단독으로 거대화는 죽은 디자인 — 의미 단위와 함께)

### 1글자만 거대 강조는 죽었다
이전 보고서에서 확인 — 우리 트라이얼의 "왜?" 1글자 거대화가 0% 성공률. 노란 강조는 *반드시 의미 단위* (2~4글자) 여야 함.

### 색 분리 mini-rule
- 한 줄 안에서 노란색은 **1~2개 단어만**. 3개 이상이면 색 의미 죽음.
- 다른 단어는 흰색 + 두꺼운 검정 스트로크 (8~14px) 가 표준.
- 빨간색은 "단 1개의 가장 강한 단어" — "충격" 또는 "통곡" 같은 결과 동사에만.

## 일본/한국 비교 영상 특화 분석

수집된 일본 관련 영상 5개 (#6, #7, #8, #9, #19, #13) 의 공통 패턴:

| 요소 | 빈도 |
|---|---|
| 일본을 **수동/패배자** 위치에 놓음 ("점령당한", "뺏긴", "발칵 뒤집힌") | 5/6 |
| 일본인의 **인용** 활용 ("살고싶어 우는 중") | 3/6 |

| 본문 안 직함 짧게 ("日 언론", "日 여성들") | 5/6 |
| 시각적 일장기 등장 (썸네일에 작은 사이즈) | 3/6 |
| 통계/숫자 동반 | 2/6 |

### 일본 비교 영상의 톤 스펙트럼
1. **공격형** (이슬람/한류 점령 — 신작가 류) — 우리 영상에 안 맞음. 학술 commentary 와 분리.
2. **동경형** ("한국에서 살고싶어 우는 중") — 우리 영상에 **딱 맞음**. 일본 여성이 한국을 동경 → 실제 이주.
3. **인용형** ("이러다 한국에 다 뺏긴다") — 일본인 본인의 한탄을 그대로 가져옴. 이게 가장 객관적이고 권위적이고 클릭률도 높음.

**핵심 인사이트:** 우리 영상은 *일본인이 직접 쓴 논문*이라는 점을 인용 톤 + 동경 톤 **둘 다** 살릴 수 있는 매우 드문 케이스. 신작가의 #9 + YTN의 #13 패턴을 융합.

### 영상의 학술 출처가 있을 때 (commentary 채널 디폴트 룰)

1. **라벨에는 4~7자 한국어** + 필요시 부제 한 줄. 영문 라벨은 채널 정체성 굳을 때까지는 자제 (혹은 한국어 부제 필수).
2. **헤드라인 1줄은 인용문 또는 통계 폭격**, 2줄은 직함 + 결과/이유 약속.
3. **저자명 / 학술지명은 *제목 본문* 또는 부제에 4~6자로 압축**. ("Kiyu Itoi" → "日 학자", "Korea Journal" → "한국 학술지", "리츠메이칸大" 만 그대로).
4. **숫자는 가능하면 변환**. "66.2%" 그대로도 좋지만 "3명 중 2명" / "5명 중 3명" 으로 풀면 직관 ↑.

### 일본/한국 비교 영상일 때

5. **일본을 수동/동경/한탄 위치에 둔다** — "한국行", "한국 없인", "한국에 뺏긴다" 류 동사.
6. **일본인 본인의 인용**을 따옴표로 첫 줄에 — reality 시그널 최강.
7. **시각적으로 일장기 + 태극기 좌우 분할** 또는 작은 코너 아이콘. 일장기 너무 크면 일본 영상으로 오인.

### 어떤 영상에서도 피할 것

8. **1글자 거대 강조** ("왜?" 류) — 정보 0, 사망.
9. **추상 단어만으로 한 줄** ("진실", "운명") — 구체 동사/숫자/인용 1개는 반드시.
10. **8자 이상 한 줄** — 6~12자 (썸네일 가독성 한계). 줄당 10자 이내가 anchor.

### 자극어 사용 정책

11. **top-5 자극어 ("충격", "발칵", "경악", "통곡", "진짜")** 중 **1개만** 쓰기. 2개 이상 = 가짜 국뽕 시그널.
12. **결과 동사 (통곡/굴복/박살/뺏긴)** 는 사실 검증 후만. 우리 채널 commentary 톤상 과장 = 신뢰도 하락.

### 한자/영문 시각 활용

13. **한자 약자 (日, 韓, 美, 中)** 는 강조 단어로 좋음 — 한국 폰트에서 시각 무게 큼.
14. **"한국行"** 같은 한국어+한자 혼용은 ace 카드. 직관 + 시각 임팩트 둘 다.
15. **영문 (Korea Journal, BBC, NYT)** 은 권위 라벨에만, 본문 헤드라인엔 자제.
