## 추출한 공통 패턴 9개

### 1. **인용 prefix** — 따옴표로 시작
영상의 가장 강한 hook 한 줄을 따옴표로 떼서 맨 앞에 둠.
- `"한국인들은 맛에 타협이 없다" ...`
- `"한국이 또 미친걸 공개했다" ...`
- `'한국 핵추진 잠수함 승인' ...`

**왜 작동:** 클릭 전에 이미 "누가 말한 강한 문장" 을 보여줌 → 호기심 + 권위.

### 2. **~버린/~버리자/~깨부순** — 결정적 완료 동사 어미
- `눈치채버린`, `외치게 만들어버린`, `되갚아버린`, `써 버린`, `깨부순`, `바꿔버리자`, `폭로해버렸다`

**왜 작동:** 단순 과거 ("했다") 보다 "이미 끝난, 돌이킬 수 없는" 느낌. 결과의 무게.

### 3. **한자 혼용 single-char** — 시각 가중치
- `韓무기들`, `美스탠퍼드`, `美 도시전체`, `日 정부`, `日 여성`, `한국行`

**왜 작동:** 한글로만 쓰인 띠 사이에서 한자 한 글자가 시각적 anchor. 우리 영상 `日`, `한국行` 와 정확히 일치.

### 4. **결론 명사** — 항상 명사로 끝남
영상은 항상 "~이유 / ~실제 상황 / ~정체 / ~광경 / ~비밀 / ~반격" 같은 reveal 명사로 종료. 동사로 끝나면 약함.
- `... 충격적 이유`, `... 실제 상황`, `... 비밀 들통났다` (들통 = 비밀이 드러남 명사화), `... 광경`, `... 정체`, `... 반격`

### 5. **숫자 = 권위** — 막연한 "많이" 대신 정확한 수치
- `8억 전세계 한국 팬덤`, `2억4천만 넷플릭스 구독자`, `66.2%` (우리 영상)

**왜 작동:** "많은 사람들" 보다 "8억 명" 이 훨씬 구체적 → 신뢰감.

### 6. **결국/드디어/현재/이제** — discourse marker
사건이 누적되다가 마침내 도달한 임팩트라는 시간성 부여.
- `결국 전세계가 ...`, `드디어 세계 1위의 ...`, `현재 한국 국방부 ...`

### 7. **3마디 구조** — [인용/사건] + [세계 반응] + [원인/정체]
긴 제목은 사실 세 단위로 쪼개짐:
```
[BTS '아리랑' 컴백]  +  [전세계 유력 매체들 난리났다]  +  ["아리랑이라니..미쳤다"]
[8억 전세계 한국 팬덤이 움직였다]  +  [드디어 세계 1위의 '충격 반전']  +  [써 버린 한국 실제 상황]
[한국이 광복절에 벌인 놀라운 일에]  +  [일본이 제발 멈춰달라고한]  +  [이유]
```

### 8. **세계 반응 동사** — 한국 의 행동 → 세계가 반응
- `전세계가 눈치채버린`, `전세계 유력 매체들 난리났다`, `전세계 네티즌들과 평론가들 기립박수`, `美 도시전체 풍경을 바꿔버리자`
- 행위자가 한국, 반응자가 세계 — 이 비대칭이 국뽕 hook 의 core.

### 9. **자국 비판 / 자국 인정** — 일본/중국 영상 특화
- `일본이 제발 멈춰달라고한`, `일본정부가 크게 당황한`, `일본의 경제보복을 역으로 되갚아버린`
- "비판자가 일본 본인" 이라는 인지부조화 hook. **우리 트라이얼 (일본 학자가 일본 여성 분석) 와 정확히 일치.**

## 우리 트라이얼 영상에 적용

### Before (현재)
- 제목: `【일본인 학자 충격 논문】 일본 여성들, 직장 버리고 한국으로 떠난다`
- 썸네일: `직장 버리고 한국行` / `日 여성들 진짜 이유`

### After (바이스톰/쓸모왕 패턴 적용)

**YouTube 제목** (긴 형식, 3마디 + 한자 + 권위 명사):
```
직장 버리고 한국行 택해버린 日 여성들, 자국 명문대 교수가 논문으로 직접 폭로한 충격적 진짜 이유
```

3마디 분해:
- [직장 버리고 한국行 택해버린 日 여성들] — 사건 (~버린 완료 어미, 한자 혼용)
- [자국 명문대 교수가 논문으로 직접 폭로한] — 자국 비판자 + 권위 (쓸모왕 패턴 9 + 우리 차별화)
- [충격적 진짜 이유] — reveal 명사 (패턴 4)

**썸네일 텍스트** (2줄, 짧게):
- text_lines: `["日 교수가 논문으로 폭로", "여성들 한국行 진짜 이유"]`
- accent_words: `["폭로", "한국行", "진짜 이유"]`

이 조합은 두 채널의 패턴을 모두 가져오면서 우리 차별화 (학술 권위) 도 유지:
- 패턴 1 (인용 효과) — "폭로" 동사가 인용 hook 역할
- 패턴 2 (~버린/폭로) — `폭로` 명사가 완료감
- 패턴 3 (한자) — `日`, `한국行`
- 패턴 4 (reveal 명사) — `진짜 이유`
- 패턴 9 (자국 비판) — `日 교수` 가 일본 여성 분석 = 자국 인정 hook
