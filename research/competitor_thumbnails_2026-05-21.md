## 톱 성과자 공통점 (7개)

1. **반드시 얼굴이 있다.** 외국인의 놀란 표정 — 입 벌리고 눈 크게. 컷아웃해서 메인 콘텐츠 사진 옆/하단에 배치.
2. **텍스트 2줄, 약 50~70% 화면 차지.** 흰색 + 굵은 검정 스트로크 (8~14px). 한 줄당 12~20자.
3. **노란색은 핵심 단어 1~3개에만.** ("미국", "한국", "1위", "충격" 등). 같은 줄 안에서 색 대비.
4. **권위/인증 라벨 좌상단** "실제영상" / "자막뉴스" / "CNN/MBC" / "신작가" 등 - 빨간/파란 라운드 사각형 + 흰글자.
5. **태극기는 코너에 작게.** 약 80~120px 정사각, 좌상 또는 좌하. 큰 비중 차지 절대 X.
6. **시각적 갈등 신호:** 화살표 →, 빨간 원 ◯, "?!", "VS" — 두 요소 사이의 긴장을 명시.
7. **배경에 항상 실사 사진 또는 합성.** 그라데이션/단색 = 시각적 사망 (예외: 풀 일러스트 일관성 채널만).

## 우리 현재 썸네일 vs 경쟁작 — 갭 분석

### 우리가 빠뜨린 결정적 요소 (top-5)

1. **얼굴 0개** — 모든 톱 성과자는 얼굴이 있다. 외국인 놀란 표정이 클릭 트리거 #1.
2. **권위/인증 라벨이 없음** — "실제영상", "1차자료", "분석" 등 좌상단 빨간 라벨이 톱 채널의 26%에 있음. "파이널K" 라는 채널 워드마크는 있지만 콘텐츠 라벨 ("심층분석" 같은)은 없다.
3. **태극기가 4배 크다** — 우리 태극기는 약 340x227px (썸네일의 약 18%). 톱 채널은 80~120px (약 3~5%) 코너에만.
4. **텍스트 정보가 너무 적다** — "왜?" 1글자 + "직장 버리고 한국행" 보조. 톱 채널은 2줄에 두 가지 다른 정보 제공 (인용문 + 결과/이유).
5. **시각적 갈등 신호 0** — 화살표, 빨간 원, VS, "?!" — 5장 중 평균 1~2개가 있음. 우리는 "?" 만 있는데 그건 본문 글자라서 강조 마커가 아니다.

### 우리가 잘못 적용한 요소

1. **거대한 노란 단어** — "왜?" 가 화면의 약 60%를 차지. 톱 채널 어디에도 이 비율 없음. 1글자만 거대화는 시각적 외침이지만 정보가 0이라 호기심 형성 실패. 톱 채널은 큰 글자도 "한국 1위" "일본 충격" 처럼 의미 단위.
2. **단순 그라데이션 배경** — 50장 중 약 4장(8%)만 그라데이션/단색이고 그것들은 채널 일러스트 일관성 차원에서 의도된 것. 우리는 stock 사진을 안 깐 fallback 으로 그라데이션이 되어버려서 "정보 없는 빨간 화면"이 됐다.
3. **서브카피가 너무 가는 글자, 너무 작게** — 약 60px Bold. 톱 채널 보조 텍스트도 100~130px Black 굵기. 한 줄도 너무 얇아서 사진/일러스트 위에서 가독성 잃기 쉽다.

## PIL 구현 가능한 개선안 (우선순위 순)

> 제약: AI 이미지 생성 NO, 유료 API NO, PIL 만 사용. 단 사용자가 외부 무료 stock photo URL 1장 넘겨주면 PIL paste/mask 합성 가능.

---

### 개선안 1: 좌우 분할 비교 컴포지션 (Split Screen)

- **무엇:** 썸네일을 좌우 50:50 으로 가르고 좌측에는 stock A (예: 외국인 사진), 우측에는 stock B (예: 서울 도시). 가운데에는 노란 "VS" 또는 노란 세로 사선.
- **어떻게:**
  - `Image.paste(stock_a, (0,0), mask=mask_left)` 와 `Image.paste(stock_b, (W//2,0), mask=mask_right)`
  - 두 영역 사이에 `ImageDraw.polygon` 으로 노란 사선 또는 흰 세로 디바이더
  - 중앙에 `font.getbbox("VS")` 로 노란 큰 글자 + 검정 스트로크
- **어디:** `build_thumbnail.py` 에 `--bg-left`, `--bg-right` 두 인자 추가 + `background.type = "split"` spec 지원. youtube-thumbnail-brief 스킬은 비교 영상일 때 split mode 추천하도록 룰 추가.
- **비용:** 구현 4~6시간. stock 사진 2장 필요 (사용자 또는 stock-query-tagger agent 가 Pexels URL 2개 넘김).
- **임팩트:** 상 — 비교 콘텐츠 thumbnails(군사력/문화/before-after) 의 시그니처. 한일/한미 비교 영상이라면 거의 필수.

---

### 개선안 2: 좌상단 권위/인증 라벨 박스

- **무엇:** 우리 영상마다 "심층분석" / "1차자료" / "해외 학계" 같은 4~6자 라벨을 좌상단에 빨간 라운드 박스 + 흰글자로.
- **어떻게:**
  - `ImageDraw.rounded_rectangle((20, 20, 200, 75), radius=8, fill=(220, 38, 38))`
  - 흰글자 32~38px Bold 중앙 정렬
  - 살짝 그림자: `ImageFilter.GaussianBlur` 로 박스 복제 → 검정 → blur → 그 위에 원본
- **어디:** `build_thumbnail.py` 에 `--label "심층분석"` 또는 `spec.label_text` 필드. 모든 영상에 기본값 부여.
- **비용:** 구현 1시간. 새 입력 없음 (스크립트 작성시 토픽 따라 자동 결정 가능).
- **임팩트:** 상 — 채널 브랜드 인식 + 콘텐츠 카테고리 신호. 26% 톱 채널이 사용중.

---

### 개선안 3: 태극기를 코너 작은 아이콘으로 축소

- **무엇:** 현재 340x227 태극기를 96x64 로 축소하고 좌상단으로 이동 (또는 라벨 옆). 흰 테두리는 유지.
- **어떻게:** `draw_korean_flag(96, 64)` 호출, 위치 `(label_right + 12, 24)`. 라벨 없을 때는 단독으로 (20, 20).
- **어디:** `build_thumbnail.py` 의 flag size constants 와 위치 로직.
- **비용:** 30분 (이미 함수 파라미터화 되어 있음).
- **임팩트:** 중-상 — 화면 점유율 18% → 3% 로 줄여서 텍스트 공간 확보. 또한 톱 채널 컨벤션 일치.

---

### 개선안 4: 큰 비주얼 펑처에이션 ("?!" / "!?" / "...")

- **무엇:** 메인 텍스트 옆 또는 인물 옆에 큰 빨간색/노란색 `?!` `!?` `!!!` 를 stroke 굵게.
- **어떻게:** 별도 컬러로 `d.text((x,y), "?!", font=font_big, fill=YELLOW, stroke_width=14, stroke_fill=BLACK)`. 살짝 회전하면 더 다이내믹 — `Image.new` 위에 그려서 `rotate(-8)` 후 paste.
- **어디:** `build_thumbnail.py` 새 함수 `draw_punctuation_accent(img, x, y, text, color)`. spec 의 `accent_punct: "?!"` 필드로 옵션화.
- **비용:** 1.5시간 (rotation 합성 포함).
- **임팩트:** 중 — 30% 톱 채널이 사용. 시각적 호기심 트리거.

---

### 개선안 5: 풀-블리드 stock 사진 + 검정 그라데이션 띠 (하단)

- **무엇:** 배경 stock 사진을 thumbnail 전체에 깔고, 하단 40% 에 검정→투명 그라데이션 띠를 올려서 텍스트 가독성 확보. 현재 코드에 일부 있지만 강화 필요.
- **어떻게:**
  - 이미 `extract_frame` 으로 stock clip 첫 프레임 추출 로직 존재 — 작동 잘 됨
  - 하단 그라데이션 강화: alpha 0 → 220 (현재 140), 그라데이션 시작점 H*0.40 (현재 0.25 보다 아래 시작해서 텍스트 영역만 명확히 어둡게)
  - 사진 자체에 살짝 saturation boost: `ImageEnhance.Color(img).enhance(1.15)`
- **어디:** `build_thumbnail.py` 의 `bg_darken` 로직 위쪽 그라데이션 부분 수정.
- **비용:** 30분.
- **임팩트:** 상 — 그라데이션 fallback 을 사실상 제거. stock 사진이 없으면 빌드 실패시키는 게 차라리 낫다.

### 개선안 6: 텍스트를 2줄로 강제 분할 + 컬러 강조어

- **무엇:** 현재 main_copy 1줄(때로 자동 줄바꿈) → 항상 2줄로 분할하고, 각 줄에서 키워드 1개를 노란색으로 색칠.
- **어떻게:**
  - 스크립트가 spec.text 를 `["일본인 여성이", "한국 이민 가는 이유"]` 처럼 2줄 배열로 넘김
  - 각 줄을 fit_font 로 따로 자리잡고
  - 키워드 인덱스 받아서 ImageDraw 가 단어 단위로 fill 색 바꿔서 그리기:
    ```python
    def draw_text_with_accent(d, text, font, x, y, accent_word, accent_fill, base_fill):
        # split by space, draw each chunk with proper fill
    ```
- **어디:** `build_thumbnail.py` 메인 텍스트 그리기 부분 전면 리팩토링. spec 에 `text_lines: ["..", ".."]` 과 `accent_words: ["일본", "한국"]` 필드 추가.
- **비용:** 4시간 (단어 너비 계산 정밀하게).
- **임팩트:** 상-상 — 정보 밀도 2배 증가 + 색 대비. 54% 톱 채널이 이걸 한다.

### 개선안 7: 외국인 얼굴 컷아웃 (사용자 입력 1장 + PIL 마스킹)

- **무엇:** 사용자가 free stock photo URL 1장 (외국인 놀란 표정) 넘기면, PIL 로 가장자리 페더링하고 우측 하단에 paste.
- **어떻게:**
  - Pexels API 무료 (사용자 토큰) 또는 stock-query-tagger agent 가 "shocked foreigner face" 같은 검색어로 URL 추천
  - 받은 사진을 정사각으로 crop → 원형 또는 둥근 사각형 마스크로 잘라내기:
    ```python
    mask = Image.new("L", (size, size), 0)
    ImageDraw.Draw(mask).rounded_rectangle((0,0,size,size), radius=40, fill=255)
    mask = mask.filter(ImageFilter.GaussianBlur(6))  # soft edge
    img.paste(face, (x, y), mask)
    ```
  - 흰 테두리 추가 (rounded_rectangle outline)
- **어디:** `build_thumbnail.py` 에 `--face` 인자. youtube-thumbnail-brief 스킬에 stock photo 검색 단계 추가.
- **비용:** 3시간 (다운로드 + crop + mask + paste). 운영 비용 0 (Pexels 무료).
- **임팩트:** 상-상 — 가장 큰 갭이 얼굴 부재이므로 한 장만 추가해도 즉시 시각적 정상화. 76% 톱 채널과 동등.

### 개선안 8: 인용문 따옴표 텍스트 (외국인 발언 인용)

- **무엇:** 메인 텍스트 일부를 `"한국이 진짜 충격이네요"` 처럼 큰 따옴표로 감싸기. 영상 마다 외국인/전문가의 핵심 인용 1개를 thumbnail copy 로.
- **어떻게:**
  - 텍스트 앞뒤에 ` " ` ` " ` 를 큰 따옴표 글자로 추가 + 색은 본문과 다르게 (예: 빨간색)
  - PIL `ImageDraw.text` 가 어차피 따옴표 잘 렌더
- **어디:** youtube-thumbnail-brief 스킬: 영상 스크립트에서 핵심 외국인 발언 1줄 자동 추출하라는 룰 추가. `text_quotation: true` spec.
- **비용:** 30분 (텍스트 변환만).
- **임팩트:** 중 — 권위/리얼리티 시그널. 톱 reaction 채널 약 40% 사용.

### 개선안 9: "전후 비교" 또는 "국가 비교" 시 분할 with 국기 합성

- **무엇:** 개선안 1 의 강화형. 좌우 분할 시 각 사이드에 해당 국가 국기를 반투명으로 배경에 깔기 (한국=태극기, 일본=일장기, 미국=성조기). 이미 우리 코드에 태극기 함수 있음.
- **어떻게:**
  - 일장기: 흰 배경 + 중앙 빨간 원 — 매우 간단:
    ```python
    def draw_japan_flag(w, h):
        img = Image.new("RGB", (w, h), WHITE)
        d = ImageDraw.Draw(img)
        r = int(h * 0.3)
        d.ellipse([w//2 - r, h//2 - r, w//2 + r, h//2 + r], fill=(188, 0, 45))
        return img
    ```
  - 성조기: stripes + stars — 좀 더 복잡하지만 30줄 정도. 또는 사용자가 PNG 1장 넘기게.
  - 분할된 영역 위에 `paste(flag, ..., mask=alpha_64)` 처럼 약 25% 불투명도
- **어디:** `build_thumbnail.py` 에 `draw_japan_flag`, `draw_usa_flag` 추가. spec 의 `background.type = "split_flags"`, `flags: ["KR", "JP"]`.
- **비용:** 3시간 (한국기 외 2개 추가 + 합성 로직).
- **임팩트:** 상 — 비교 영상에서 즉시 visual identity. 톱 채널의 한일 비교 영상은 100% 이 패턴.

### 개선안 10: 가독성 보장 — 텍스트 영역 다크 박스

- **무엇:** 텍스트 줄을 그릴 때, 글자 뒤에 반투명 검정 박스 (선택사항: 또는 빨강/노랑 띠 박스) 를 깔아서 stock 배경 무관 가독성 확보.
- **어떻게:**
  - 각 텍스트 줄의 bbox 계산 + 약간 패딩
  - `Image.new("RGBA", (W, H), 0)` 위에 `ImageDraw.rounded_rectangle(padded_bbox, fill=(0,0,0,180))`
  - 그 위에 텍스트 합성
- **어디:** `build_thumbnail.py` 텍스트 그리기 직전. spec 의 `text_band_fill_hex`, `text_band_alpha` 옵션.
- **비용:** 1.5시간.
- **임팩트:** 상 — 스트로크 14px + 다크 박스 조합이 가장 안정적 가독성. 22% 톱 채널이 사용. stock 배경 도입과 패키지로 묶어야 함.

### 개선안 11: 빨간 원 ◯ / 화살표 → 강조 마커

- **무엇:** 사진 안 특정 영역에 빨간 동그라미를 그려서 "여기 봐라" 마커. 또는 빨간 화살표가 텍스트에서 이미지로 지시.
- **어떻게:**
  - `ImageDraw.ellipse([x, y, x+r, y+r], outline=RED, width=8)` 빈 원
  - 화살표는 line + 폴리곤 삼각형
  - 위치/대상은 spec 에서 명시 (`accent_circle: {x: 0.6, y: 0.4, r: 80}`)
- **어디:** `build_thumbnail.py` 새 함수 `draw_attention_marker`. spec 에 `markers: [...]` 배열.
- **비용:** 1.5시간.
- **임팩트:** 중 — 24% 톱 채널이 사용. stock 사진을 깔 때만 의미 있음.

### 개선안 12: 채널 워드마크 위치/스타일 통일

- **무엇:** 우리 "파이널K" 가 현재 우하단에 노란 글자로 박혀있는데 너무 작고 brand identity 약함. 톱 채널들은 좌상단 또는 우상단에 약 80x80 워드마크 + 로고 이미지를 함께 쓴다.
- **어떻게:**
  - "파이널K" 둥근 사각 박스 안에 (배경 검정 / 노란 글자 / 흰 테두리) — 80px 정사각 가까이
  - 위치: 우상단 (태극기를 좌상단으로 옮긴 후 자리 비움)
- **어디:** `build_thumbnail.py` 마지막 brand 블록. spec 의 `wordmark_style: "badge"`.
- **비용:** 1시간.
- **임팩트:** 중 — 채널 정체성 강화. 시리즈 영상에서 누적 임팩트 큼.

## 부록: 인상적인 썸네일 5장 케이스 스터디

### A. 국뽕에 취하다 "BTS ON 이정도였어?" (y1uCAfjIa30)
- **왜 강력한가:** 상단 BTS 댄스 컷 → 하단 외국인 리액터 12명의 입벌린 표정 콜라주 → 본문은 인용문 1줄 + 결과 질문 1줄. 정보 밀도, 감정 시그널, 권위 인증("실제영상" 빨간 라벨) 모두 만점.
- **우리가 빌릴 요소:** 좌상단 빨간 인증 라벨, 텍스트 2줄 + 노란 강조어, "한/영자막" 같은 마이크로 뱃지.

### B. 지식스토리 "2026 군사력 비교 한국 VS 일본" (wRA-XLrCWyk)
- **왜 강력한가:** 풀스크린 태극기/일장기 분할 배경, 중앙 큰 VS, 양쪽에 한국군/일본군 인물 컷아웃, 가운데 화염 효과. 비교 영상의 정석. 텍스트가 적은데도 한눈에 비교 영상임을 알 수 있다.
- **우리가 빌릴 요소:** 좌우 분할 + 국기 합성 + VS + 양쪽 stock 컷아웃 — 우리 비교 영상에 즉시 적용 가능.

### C. "한국 5위... 진짜냐?" (mX1w8CMogU8)
- **왜 강력한가:** 화면 정중앙에 **빨간 박스 안 흰글자 "일본도 놀랐다"** + **노란 박스 안 검정글자 "한국 5위 진짜냐?"** 적층. 박스가 콘텐츠를 압도하지만 정보가 충격적이라 클릭 유발.
- **우리가 빌릴 요소:** 빨강/노랑 텍스트 띠 박스 적층 디자인. 정보 폭격형 디자인의 본보기.

### D. "1950 vs 2020 한국" (3Kzj2-JnQhk)
- **왜 강력한가:** 좌측 흑백 1950 한국 폐허 vs 우측 2020 서울 야경 도시 — before/after 시각적 충격. 하단 검정 띠에 흰글자 3줄. 시간 이동의 시각적 비유 완벽.
- **우리가 빌릴 요소:** Before/after 분할. 우리 K-culture 콘텐츠에서 과거/현재 비교 영상에 즉시 적용.

### E. "트럼프 말도 안 돼! 서울에 트럼프가? 충격!" (LV7g4ko3Rkg)
- **왜 강력한가:** 우측에 트럼프 컷아웃 + 노란 말풍선 "말도 안 돼!" / 좌측에 청계천 야경 / 하단 큰 글자. 유명 인물 + 표정 + 말풍선이 컬튜럴 hook. **PIL 로 가장 쉽게 카피 가능**한 디자인 — stock 사진 1장 + 말풍선 1개만 추가하면 됨.
- **우리가 빌릴 요소:** 우측 인물 컷아웃 + 말풍선. 개선안 #7 + 새 개선안 (말풍선 도형 그리기) 결합.
